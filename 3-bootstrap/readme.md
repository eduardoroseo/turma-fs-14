# Bibliotecas CSS

## O que são?

Bibliotecas CSS são conjuntos de estilos prontos para serem utilizados em projetos web. Elas são úteis para agilizar o desenvolvimento de telas, pois fornecem uma série de componentes prontos para serem utilizados.

## Por que usar?

Para agilizar o desenvolvimento de telas, evitando a necessidade de criar estilos do zero.

## Quais são as principais?

- [Bootstrap](https://getbootstrap.com/)
- [Bulma](https://bulma.io/)
- [Materialize](https://materializecss.com/)
- [Tailwind](https://tailwindcss.com/)

## Tem mais alguma?

Sim, existem muitas outras bibliotecas CSS disponíveis. Algumas são mais genéricas, enquanto outras são mais específicas para determinados tipos de projetos. Além disso, algumas são mais populares, enquanto outras são menos conhecidas.

## Como escolher?

A escolha da biblioteca CSS depende do projeto e do gosto pessoal do desenvolvedor. Além disso, é importante considerar a popularidade da biblioteca, pois isso pode facilitar a busca por soluções para problemas específicos.

## Qual eu considero a mais bonita?

Apesar de ser uma questão de gosto pessoal, acredito que a biblioteca [Tailwind](https://tailwindcss.com/) é a mais bonita. Ela possui um design moderno e minimalista, que se destaca em relação às outras bibliotecas.

## Qual eu considero a mais fácil de usar?

Acredito que a biblioteca [Bootstrap](https://getbootstrap.com/) é a mais fácil de usar. Então vamos aprender a utilizá-la!

# Bootstrap

## O que é?

[Bootstrap](https://getbootstrap.com/) é uma biblioteca CSS que fornece uma série de estilos prontos para serem utilizados em projetos web. Ela é útil para agilizar o desenvolvimento de telas, pois fornece uma série de componentes prontos para serem utilizados.

## Qual é a versão mais recente?

A versão mais recente do [Bootstrap](https://getbootstrap.com/) é a 5.0.

## Como instalar?

Para instalar o [Bootstrap](https://getbootstrap.com/), basta adicionar o seguinte código no cabeçalho do HTML (dentro da tag `<head>`):

```html
<link
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
  rel="stylesheet"
  integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH"
  crossorigin="anonymous"
/>
```

E adicionar o seguinte código no final do HTML (antes da tag `</body>`):

```html
<script
  src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
  integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz"
  crossorigin="anonymous"
></script>
```
